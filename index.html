Accomplishment Stories

Deep Reinforcement Learning for WiFi Optimization

Situation: The Wireless IoT Lab needed to optimize WiFi-6 network performance for VR applications, but traditional approaches weren't adapting well to dynamic channel conditions.

Task: Design and implement a deep reinforcement learning system that could dynamically optimize channel access in real-time.

Action: I architected a Deep Q-Network agent using PyTorch with custom reward shaping and experience replay mechanisms, iterating through multiple configurations over four months.

Result: Achieved 25% performance improvement. Published at IEEE COMSNETS 2025 and received Best Undergraduate Presentation Award.

Production RAG Pipeline at Pibit.ai

Situation: Pibit.ai was struggling with manual prompt engineering processes taking days of manual testing with inconsistent results.

Task: Build a production-ready RAG pipeline that could automate prompt optimization and significantly reduce iteration time.

Action: Designed complete RAG pipeline using Langchain with vector embeddings, integrated DSPy for automation, and deployed on Azure with MLflow tracking.

Result: Reduced manual iteration time by 60% and achieved 40% accuracy improvement. Pipeline deployed to production processing thousands of queries daily.

Cloud-Native MCP Server at Northeastern

Situation: The Wireless IoT Lab needed a scalable system for equipment reservations that could integrate with Claude AI for natural language interactions.

Task: Architect and deploy a cloud-native Model Context Protocol server with multi-user authentication and LLM integration.

Action: Designed microservices architecture on OpenShift/Kubernetes, built dual-proxy gateway for LLM tool-calling, implemented JWT/LDAP auth, and established CI/CD pipeline.

Result: Production system now used by 50+ lab members handling 200+ reservations monthly with zero conflicts. Became reference implementation for LLM integration.
