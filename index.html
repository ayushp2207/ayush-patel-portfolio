Accomplishment Stories

Deep Reinforcement Learning for WiFi Optimization

Situation: The Wireless IoT Lab needed to optimize WiFi-6 network performance for VR applications, but traditional approaches weren't adapting well to dynamic channel conditions. As a thesis scholar, I was tasked with developing an intelligent solution.

Task: Design and implement a deep reinforcement learning system that could dynamically optimize channel access in real-time, improving network performance for latency-sensitive VR traffic while maintaining overall quality of service.

Action: I architected a Deep Q-Network (DQN) agent from scratch using PyTorch, implementing custom reward shaping mechanisms to balance throughput and latency. I developed an experience replay system to stabilize training and created a comprehensive testing framework using MATLAB simulations. Over four months, I iterated through multiple reward function designs and hyperparameter configurations.

Result: Achieved 25% performance improvement over baseline methods. The research was published at IEEE COMSNETS 2025 and received the Best Undergraduate Presentation Award. The solution is now being considered for real-world deployment in the university's wireless infrastructure.

Production RAG Pipeline at Pibit.ai

Situation: During my internship at Pibit.ai, the company was struggling with manual prompt engineering processes that were taking significant time and yielding inconsistent results for their document retrieval system. Each iteration required days of manual testing.

Task: Build a production-ready Retrieval-Augmented Generation pipeline that could automate prompt optimization and significantly reduce the time needed for iterations while improving accuracy.

Action: I designed and implemented a complete RAG pipeline using Langchain with vector embeddings for semantic document retrieval. I integrated the DSPy framework to automate prompt optimization, eliminating manual trial-and-error. I also engineered advanced prompting strategies including Few-Shot learning, Chain-of-Thought reasoning, and ReAct patterns. Finally, I deployed the system on Azure with MLflow for experiment tracking and set up A/B testing infrastructure.

Result: Reduced manual iteration time by 60% and achieved 40% accuracy improvement over the baseline system. The pipeline was deployed to production and is now processing thousands of queries daily. The framework I built became a template for future ML projects at the company.

Cloud-Native MCP Server at Northeastern

Situation: The Wireless IoT Lab at Northeastern needed a scalable system for equipment reservations that could integrate with Claude AI for natural language interactions. The existing system was manual and prone to scheduling conflicts.

Task: Architect and deploy a cloud-native Model Context Protocol server that could handle multi-user authentication, integrate with LLMs for natural language processing, and provide reliable equipment reservation capabilities.

Action: I designed a microservices architecture deployed on OpenShift/Kubernetes, implementing containerized services with Docker. I built a dual-proxy gateway that enabled Claude AI to perform tool-calling for equipment reservations through a RESTful API. I implemented secure multi-user authentication using JWT tokens and LDAP integration, along with MySQL-backed session management. I also established a CI/CD pipeline for automated testing and deployment.

Result: Successfully deployed a production system now used by 50+ lab members for equipment scheduling. The system handles 200+ reservations per month with zero scheduling conflicts. The architecture has become a reference implementation for other labs considering LLM integration. The project showcased the practical application of Model Context Protocol in academic settings.
